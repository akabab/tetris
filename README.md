# tetris


### Pre-required

install [n](https://github.com/tj/n) (node version manager)
```
$ curl -L https://git.io/n-install | bash
```
install latest version of node
```
$ n latest
```

### Installation

clone repo
```
$ git clone https://github.com/akabab/tetris && cd tetris
```

install dependencies
```
$ npm install
```

### Development Mode

#### Launch Server

```
$ npm run srv-dev
```

#### Launch Client

```
$ npm run client-dev
```
